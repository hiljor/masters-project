# masters-project

## Web Tool

This project includes a web-based GUI for selecting algorithms and data files to solve graph problems.

### Installation

Ensure you have the necessary dependencies installed:

```bash
pip install fastapi uvicorn
```

### Usage

To start the web server, run the following command from the project root:

```bash
python web/server.py
```

Once the server is running, open your browser and navigate to `http://localhost:8000`.

## Testing

To run the tests for this project, use the following command:

```bash
pytest
```
